# Aplikasi Cuaca (Flutter)

Aplikasi cuaca sederhana yang menampilkan kondisi cuaca terkini berdasarkan nama kota, menggunakan data dari OpenWeatherMap.

## Fitur

- Cari cuaca berdasarkan nama kota
- Menampilkan suhu, kondisi cuaca, ikon, suhu terasa, suhu min/max
- Detail tambahan: kelembapan, kecepatan angin, tekanan udara
- Background gradient yang berubah sesuai kondisi cuaca (cerah, berawan, hujan, dll)
- Tarik ke bawah (pull to refresh) untuk memuat ulang data
- Default kota awal: Banda Aceh
